# KCL Ticketing System - Frontend

## Setup

```bash
npm install
npm start
```

## Available Backend API

- **POST** `/api/submit-ticket/` - Submit a new ticket

## Project Structure

```
src/
├── App.js          - Main component (placeholder)
├── index.js        - React entry point
├── utils/          - Utility functions (add your own)
└── components/     - Add your components here
```

## Getting Started

1. Run backend: `python manage.py runserver` (port 8000)
2. Run frontend: `npm start` (port 3000)
3. Build your components in `src/`
4. API calls will proxy to `http://localhost:8000`
